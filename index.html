Enable Real-Time Sync

Zonemate uses Firebase Realtime Database (free tier) to sync changes between you and Venkatesh instantly. Setup takes ~3 minutes.

1

Go to console.firebase.google.com and sign in with Google.

2

Click Add project → name it "zonemate" → skip Google Analytics → Create project.

3

In the left sidebar: Build → Realtime Database → Create database → choose any location → Start in test mode (allows read/write for 30 days; you can extend later).

4

Go to Project Settings (gear icon) → scroll to "Your apps" → click Web (</>) → Register app → copy the firebaseConfig object.

5

Open zonemate.html in a text editor → find the firebaseConfig block near the top → paste your config values replacing the YOUR_... placeholders → save.

6

Share the updated zonemate.html file with Venkatesh. Both of you open it — changes appear instantly for both. 🎉

The free Spark plan supports 100 simultaneous connections and 1 GB storage — more than enough for this app forever.
